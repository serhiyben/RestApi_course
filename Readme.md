## Що реалізовано в Лабі №9 Навантажувальне тестування з Locust
Мета роботи — перевірити продуктивність та стабільність API під навантаженням, визначити пропускну здатність (RPS) та час відповіді (Response Time).


### Основні зміни та досягнення (Лабораторна робота №8):

Було проведено два етапи тестування: Load Test (нормальне навантаження) та Stress Test (підвищене навантаження).

# Load Test (50 одночасних користувачів)
Користувачів: 50
Швидкість приросту (Spawn Rate): 5 юзерів/сек
Результат (RPS): ~29.1 запитів/сек
Середній час відповіді: 3 мс
Помилки (Failures): 0%

# Stress Test (100 одночасних користувачів)
Користувачів: 100
Швидкість приросту (Spawn Rate): 10 юзерів/сек
Результат (RPS): 56.6 запитів/сек
Середній час відповіді: 3 мс
95-й перцентиль (Max Response Time): 5 мс
